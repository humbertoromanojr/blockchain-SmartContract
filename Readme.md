# Smart Contract Technical Challenge: Advanced Yield Streamer

### Setting up your Workspace
1. Run `npm install` to download project infrastructure dependencies.
2. Compile your workspace contracts using:
   ```bash
   npm run compile
   ```

### The Evaluation Track
Your task is to open `contracts/YieldStreamer.sol` and fulfill the baseline implementation markers marked with `TODO`. 

### Testing and Formatting Requirements
Your solution will be assessed based on terminal output integrity. You must write complete test assertions matching the framework templates inside `test/YieldStreamer.test.ts`. 

To process and view the active status of your implementation assertions on your local terminal, execute:
```bash
npm run test
```
